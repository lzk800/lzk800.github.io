1、统计出/etc/passwd文件中其默认shell为非/sbin/nologin的用户个数，并将用户都显示出来

  • 统计shell为非/sbin/nologin用户的个数：

    ~]# cat /etc/passwd | grep -v '/sbin/nologin'|wc -l

  • 显示用户：

    ~]# cat /etc/passwd | grep -v '/sbin/nologin'|cut -d: -f1
